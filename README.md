This repository exists solely to host **GitHub Discussions** for the blog comment system on [Md Mohsin Hossain's academic portfolio](https://mdmohsinhossain.github.io).

## How it works

1. A visitor reads a blog post at `mdmohsinhossain.github.io/blog/…`
2. At the bottom of each post, a giscus widget loads and maps the post URL to a Discussion in this repository (using the **Announcements** category).
3. Visitors can react or comment by signing in with their GitHub account.
4. Discussions are threaded, searchable, and fully moderated through GitHub's standard interface.

No tracking, no cookies, no third-party ads — just GitHub.

---

## Moderation

All new Discussions land in the **Announcements** category, which is write-protected by default — only repository maintainers can open new threads. This prevents spam while still allowing anyone with a GitHub account to reply.

To moderate comments:

- Visit the [Discussions tab](../../discussions) in this repository.
- Use GitHub's built-in tools to pin, lock, edit, hide, or delete any comment.
- To pin a Discussion (e.g. a sticky note), mark it as **Pinned** from the Discussion's menu.

---

## Configuration

The giscus widget is configured in `src/config.ts` of the main portfolio repository and rendered in `src/pages/blog/[...slug].astro`. Key settings:

| Setting           | Value                              |
| ----------------- | ---------------------------------- |
| Repository        | `mdmohsinhossain/blog-comments`    |
| Discussion category | Announcements                    |
| Mapping           | `pathname` (strict)                |
| Reactions         | Enabled                            |
| Lazy loading      | Enabled                            |
| Theme             | Synced with site light/dark toggle |
